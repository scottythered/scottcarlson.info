Professional Service, Activities, & Projects
Author, Exiegfried (2019)
A Python program to auto-generate EXIF and signature-based file format identification metadata of Archival Information Packages created by the Woodson Research Center at Rice University's Fondren Library. Constructed as both a stand-alone script or a GUI-based app using the Gooey library.

Author, Asseta Code (2019)
An ongoing repository of various code work during my time of employment at Asseta.com.

Author, ArchivesSpace Reporting and Cleanup Projects (2017-2019)
An ongoing collection of Python scripts developed to interact with local ArchivesSpace instance using the ArchivesSpace REST API and the ArchivesSpaceSnake library.

Author, Alma P2E ISSN/Fuzzy Title Matching (2018-2019)
Data cleanup project developed in preparation for migration to the Ex Libris Alma platform. Alma requires tradition MARC records for electronic resources to be converted to its different database structure; this notebook matches between MARC bib records and SFX holdings via ISSN and fuzzy title matches.

Technical Lead, DLF-AIG Metadata Group's MAP Clearinghouse Project (2017-2019)
The MAP Clearinghouse is a website that collects metadata application profiles, mappings, procedures, and descriptive metadata strategies in digital repository collections. The site was built by modifying an existing project using NodeJS.

Organizing Committee Member, New Librarian Summit Conference (2018) and Personal Digital Archiving Conference (2018)

Author, Rice Digital Scholarship Archive Analysis Using Scopus References (2018)
A data analysis project utilizing the Scopus institutional API to cross-reference scholarly works that cite materials deposited in the Rice Digital Scholarship Archive (RDSA). The analysis indicates that some works are probably not being correctly cited as originating from the RDSA.

Author, You Should Make a Twitter Bot (2017)
A tutorial on using existing metadata to expose institutional collections to wider audiences on social networking platforms with Twitter bots.

Member, ALCTS Monographs Editorial Board (2016 - 2018)
The Monographs Editorial Board is charged with supporting the series editor in acquiring and reviewing appropriate manuscripts.

Co-Chair, ALCTS/LITA Library linked Data Interest Group (2017 - 2019)
Goals include raising awareness of semantic web technologies, promoting research on linked data challenges, and informing the ongoing development of existing metadata standards for libraries, archives and cultural heritage institutions.

Description Coordinator, Data Rescue Houston (February - March 2017)
Member of the coordinating team for the Houston Data Rescue event held at Fondren Library, Rice University, on March 4, 2017.

Author, Grateful Data (2016)
Grateful Data is a GitHub-hosted tutorial on data acquisition, cleaning, and enriching using publicly accessible data of the Grateful Dead.

Author, Cleaning Messy Data With OpenRefine (2015)

Founder, Frodis Records (2015 - 2017)
Founder of Frodis Records, an archival record label specializing in vinyl-only releases of obscure power pop and punk.

Co-Founder, Indie Preserves (2015 - 2017)
Co-founder/contributor of Indie Preserves, a website that provides indie labels and bands with physical and digital preservation advice. 

Archiving Intern, Grateful Dead Archive (July, 2013)
130 hours of volunteer work processing (appraisal, arrangement and description) the Jerry Garcia Memorial Altar Collection housed at UC-Santa Cruz, California.
